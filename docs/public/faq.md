# FAQ

## Why is there no download page yet?

We are keeping the public site focused on product information, feedback, and roadmap first.

## How should users give feedback?

Use GitHub Issues for bugs and feature requests. Include version, environment, and logs when possible.

## What will the roadmap show?

Only public-safe items that can be shared without overcommitting to dates or internal implementation details.

## Is this a trading advice product?

No. It is a research and decision-support workspace for analysis, review, and data validation.
